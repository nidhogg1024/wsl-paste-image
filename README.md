# 终端贴图助手 - uTools 插件

截图后自动转换为 WSL 路径，方便在终端中粘贴图片路径。

## 使用场景

在 WSL 终端（如 Cursor AI 终端）中，无法直接粘贴图片。本插件：
1. 截图 → 保存到临时目录 → 转为 WSL 路径 → 自动复制
2. 在终端中 Ctrl+V 粘贴路径，AI 即可读取图片分析

## 功能

- **截图转路径**：自动检测 PixPin / Snipaste / ShareX，未安装则使用系统截图
- **剪贴板转路径**：将已有的剪贴板图片转为 WSL 路径
- 路径自动复制到剪贴板，Ctrl+V 即可粘贴
- 自动清理超过 1 天的临时文件

## 安装

1. 在 uTools 中搜索「终端贴图」或「截图转路径」
2. 或下载离线包手动安装

## 使用

- 在 uTools 中输入「终端贴图」或「截图转路径」触发
- 点击「截图」按钮或触发后自动截图
- 截图完成后 WSL 路径自动复制到剪贴板
- 到 WSL 终端 Ctrl+V 粘贴
